\# Security Policy



WFSL Eco Engine is designed to refuse unsafe execution by default.



\## Reporting Issues

If you discover a vulnerability or a way to bypass:

\- policy enforcement

\- determinism checks

\- transcript guards



please report privately.



Do not open public issues for security concerns.



\## Guarantees

\- Deny-by-default execution

\- Deterministic output

\- Tamper detection

\- Optional cryptographic evidence
